## Installation

1.  Clone the repository:

    ```bash
    git clone <repository-url>
    cd spacex-launch-project
    ```

2.  Install dependencies:

    ```
    npm install

    ```

3.  Start the development server:

    ```
    npm run dev
    ```

4.  Launch the app :

    ```
    ctrl+click on the localhost:3000
    ```
